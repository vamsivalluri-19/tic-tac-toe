# 🎮 Tic-Tac-Toe Game

A simple and responsive Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## 🚀 Features

* Two-player gameplay
* Winner detection
* Draw detection
* Reset game button
* New game button
* Responsive design
* Smooth hover animations
* Modern UI design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Project Structure

```bash
tic-tac-toe/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/vamsivalluri-19/tic-tac-toe.git
```

2. Open the project folder

```bash
cd tic-tac-toe
```

3. Open `index.html` in your browser
